# Documentation

This folder contains the user manuals and setup guides for ONMD-OpenFlexure.

## Available manuals
- **ONMD Onboard User Manual**  
  `ONMD Onboard User Manual.pdf`  
  Instructions for onboard acquisition and analysis using the OpenFlexure microscope and Raspberry Pi.

- **PyONMD User Manual**  
  `PYONMD User Manual.pdf`  
  Detailed description of the offline ONMD analysis pipeline and visualization tools.
